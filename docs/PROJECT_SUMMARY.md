# 企业级CRM系统 - 项目总结

## 项目概述

本项目是一个专为IT企业(软件开发、系统集成、IT专业服务)设计的企业级CRM系统,支持从线索到回款的全流程管理,特别强化项目维度管理,可跟踪从售前到交付的完整项目里程碑。

## 部署说明

### 开发环境
```bash
# 启动后端
cd packages/backend
npm run start:dev

# 启动前端
cd packages/frontend
npm run dev
```

### 生产环境
```bash
# 构建前端
cd packages/frontend
npm run build

# 构建后端
cd packages/backend
npm run build

# 使用Docker部署
docker-compose up -d
```
